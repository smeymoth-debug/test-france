Unité 9 — Vacances & Nature Grammar
Passé composé, holidays, nature, countryside vocabulary

Le Passé Composé — Complete Rules

Past tense for completed actions

Formula: Subject + Auxiliary + Past Participle
-ER → -é : manger → mangé
-IR → -i : finir → fini
-RE → -u : vendre → vendu
AVOIR (most verbs):
J'ai mangé — I ate
Tu as visité — You visited
Nous avons vu — We saw

🏠 La Maison d'Être — 17 verbs using ÊTRE

aller / venirpartir / arriver sortir / rentrernaître / mourir monter / descendreentrer / retourner passer / restertomber / devenir + all reflexive verbs
With ÊTRE, past participle agrees with subject: Elle est partie, Ils sont arrivés
